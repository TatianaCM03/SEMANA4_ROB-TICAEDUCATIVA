Día 3: Creación de circuito básico de movimiento con motores DC.

2. Movimiento con Motores DC

2.1 Fundamentación teórica 

Qué son los motores DC

Los motores DC (motores de corriente directa) son dispositivos electromecánicos que convierten la energía eléctrica de corriente continua en energía mecánica, generando movimiento rotatorio. Estos motores son ampliamente utilizados en aplicaciones donde se requiere un control preciso de velocidad y dirección.

Características principales:


1. Funcionamiento simple: Operan mediante la interacción de un campo magnético y una corriente eléctrica en un conductor, produciendo torque.
2. Velocidad ajustable: La velocidad del motor puede controlarse variando la tensión o corriente que se le suministra.
3. Fácil inversión de giro: Cambiar la polaridad de la corriente en el motor permite invertir la dirección de rotación.


  • Aplicaciones comunes:
  • Robótica.
  • Ventiladores y bombas.
  • Juguetes motorizados.
  • Electrodomésticos como licuadoras y taladros.


Qué son los controles PWM

Los controles PWM (Modulación por Ancho de Pulso) son una técnica utilizada para regular la cantidad de energía suministrada a una carga mediante la modulación del ciclo de trabajo de una señal digital. En otras palabras, el control PWM ajusta la proporción de tiempo en que una señal está en alto (ON) frente al tiempo en que está en bajo (OFF), dentro de un periodo constante.

  • Principio básico:
  • Ciclo de trabajo (Duty Cycle): Es el porcentaje del tiempo que la señal está en alto durante un ciclo completo.
  •     100%: Señal completamente ON.
  •     50%: Señal ON la mitad del tiempo.
  •      0%: Señal completamente OFF.

EJEMPLO PRÁCTICO

Conexión del L298N
Paso 1: Conecta el módulo L298N al Arduino y a los motores siguiendo las conexiones indicadas:

IN1 → Pin 9 del Arduino.
IN2 → Pin 8 del Arduino.
IN3  → Pin 7 del Arduino.
IN4  → Pin 6 del Arduino.
ENA → Pin 5 del Arduino (para controlar la velocidad del motor A).
ENB → Pin 3 del Arduino (para controlar la velocidad del motor B).


Paso 2: Conecta los motores DC:

Motor A: Conecta las salidas OUT1 y OUT2 del L298N a los terminales del primer motor.
Motor B: Conecta las salidas OUT3 y OUT4 del L298N a los terminales del segundo motor.


Paso 3: Alimenta el módulo L298N:

Conecta la entrada de 12V a la entrada 12V del L298N (usa una fuente externa de 6V-12V).
Conecta GND del L298N a GND del Arduino.
Conecta el pin 5V del L298N al pin 5V del Arduino (esto alimentará el Arduino).


Paso 4: Esquema de conexiones del Circuito

ACTIVIDAD 1

Actividad 1

Objetivo: Aprender a controlar la velocidad de los motores utilizando un Arduino, girando los motores tanto hacia adelante.

Instrucciones:
 1.Conecta los motores al Arduino según el diagrama de conexiones que se te proporcionó.
2.Abre el IDE de Arduino y copia el siguiente código:

// Pines de conexión

#define IN1 9  // Dirección Motor A

#define IN2 8  // Dirección Motor A

#define IN3 7  // Dirección Motor B

#define IN4 6  // Dirección Motor B


// Pines de control de velocidad (PWM)

#define speedPin1 5  // Pin PWM para motor 1

#define speedPin2 3  // Pin PWM para motor 2


void setup() {

  // Configurar pines como salidas

  pinMode(IN1, OUTPUT);

  pinMode(IN2, OUTPUT);

  pinMode(IN3, OUTPUT);

  pinMode(IN4, OUTPUT);

  pinMode(speedPin1, OUTPUT);

  pinMode(speedPin2, OUTPUT);

 // Motores hacia adelante

  digitalWrite(IN1, HIGH);

  digitalWrite(IN2, LOW);

  digitalWrite(IN3, HIGH);

  digitalWrite(IN4, LOW);


  // Control de velocidad de los motores (50% de velocidad)

  analogWrite(speedPin1, 128); // 50% de velocidad para motor 1

  analogWrite(speedPin2, 128); // 50% de velocidad para motor 2

}

void loop() {

  // Los motores giran hacia adelante a la velocidad configurada

  // El código seguirá ejecutándose de forma continua sin cambios

}

3.Verifica el funcionamiento del circuito y documenta el comportamiento de los motores.
4. Toma fotos del circuito y registra tus observaciones en un informe breve.

Actividad 2


Actividad 2

Objetivo:

Aprender a programar y controlar los motores del carro robótico para realizar giros hacia la derecha e izquierda utilizando un Arduino.

Instrucciones:

1.Preparación del Circuito:
Conecta los motores al Arduino y al driver de motores L298N según el diagrama de conexiones proporcionado.

2.Configuración del Código:
Abre el IDE de Arduino y copia el siguiente código:

// Pines de conexión

#define IN1 9  // Dirección Motor A

#define IN2 8  // Dirección Motor A

#define IN3 7  // Dirección Motor B

#define IN4 6  // Dirección Motor B

// Pines de control de velocidad (PWM)

#define speedPin1 5  // Pin PWM para motor 1

  pinMode(IN4, OUTPUT);

  pinMode(speedPin1, OUTPUT);

  pinMode(speedPin2, OUTPUT);

}

void loop() {

  // Giro hacia la derecha

digitalWrite(IN1, HIGH);  // Motor izquierdo hacia adelante

digitalWrite(IN2, LOW);

digitalWrite(IN3, LOW);   // Motor derecho hacia atrás

digitalWrite(IN4, HIGH);

analogWrite(speedPin1, 128); // Velocidad 50% para motor izquierdo

analogWrite(speedPin2, 128); // Velocidad 50% para motor derecho

delay(1000); // Duración del giro


// Pausa

detenerMotores();

  delay(1000);

  // Giro hacia la izquierda

#define speedPin2 3  // Pin PWM para motor 2


void setup() {

  // Configurar pines como salidas

  pinMode(IN1, OUTPUT);

  pinMode(IN2, OUTPUT);

  pinMode(IN3, OUTPUT);

digitalWrite(IN1, LOW);   // Motor izquierdo hacia atrás

digitalWrite(IN2, HIGH);

digitalWrite(IN3, HIGH);  // Motor derecho hacia adelante

digitalWrite(IN4, LOW);

analogWrite(speedPin1, 128); // Velocidad 50% para motor izquierdo

analogWrite(speedPin2, 128); // Velocidad 50% para motor derecho

delay(1000); // Duración del giro

  // Pausa

  detenerMotores();

  delay(1000);

}

// Función para detener los motores

void detenerMotores() {

digitalWrite(IN1, LOW);

digitalWrite(IN2, LOW);

digitalWrite(IN3, LOW);

digitalWrite(IN4, LOW);

}

7. Carga y Prueba:
Carga el programa en el Arduino y observa el comportamiento del carro robótico.
Verifica que el carro realice giros hacia la derecha e izquierda según lo programado.
8. Documentación:
Toma fotos del circuito montado y del carro en acción.
Registra tus observaciones sobre el comportamiento del carro durante los giros (estabilidad, precisión, etc.).
Escribe un informe breve que incluya las configuraciones realizadas, los resultados obtenidos y posibles mejoras identificadas.

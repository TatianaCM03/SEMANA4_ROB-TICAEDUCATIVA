Contenido 3

Día 3: Creación de circuito básico de movimiento con motores DC.

3. Unificación del Proyecto "Carro Robótico"

3.1 Fundamentación teórica 

¿Qué es un carro robótico?

El carro robótico constituye un proyecto integrador que fusiona de manera excepcional las disciplinas de electrónica, programación y robótica, ofreciendo a los estudiantes la valiosa oportunidad de aplicar, en un contexto práctico y dinámico, los conocimientos adquiridos a lo largo de su formación académica. Este tipo de proyecto no solo permite la consolidación de conceptos técnicos, sino que también favorece el desarrollo de competencias esenciales como el trabajo colaborativo, al requerir la coordinación y cooperación entre los miembros del equipo, la resolución efectiva de problemas complejos que surgen durante el proceso de creación, y la creatividad, al estimular a los estudiantes a encontrar soluciones innovadoras y eficientes para los desafíos que enfrentan, fortaleciendo así su capacidad para abordar futuros retos tecnológicos con una perspectiva integral y estratégica.

ACTIVIDAD

Materiales necesarios:

- Placa base (plástico o madera)
-2 motores DC con ruedas
-Placa Arduino Uno
-Motor Driver L298N (para controlar los motores)
-Batería de 9V o fuente de alimentación
-Cables de conexión
-Placa de pruebas (breadboard)
-Tornillos y piezas de montaje


Paso 1: Preparar la base

- Coloca los motores DC en los soportes de la base.
- Asegúrate de que las ruedas estén bien sujetas a los ejes de los motores.


Paso 2: Conectar los motores al motor driver

- Conecta los cables de los motores a los pines de salida del motor driver L298N.
- Motor A: Conecta los cables de los dos motores del lado A del L298N.
- Motor B: Haz lo mismo para el motor B en el lado B.


Paso 3: Conectar el L298N al Arduino

- Conecta los pines de control del motor driver a los pines digitales de Arduino para controlar la dirección del movimiento:
- VCC y GND del motor driver L298N se conectan a 5V y GND de Arduino.
- IN1, IN2, IN3, IN4 se conectan a los pines de Arduino:
- IN1 a un pin digital (Ejemplo: 2).
- IN2 a otro pin digital (Ejemplo: 3).
- IN3 a otro pin (Ejemplo: 4).
- IN4 a otro pin (Ejemplo: 5).


Paso 4: Fuente de alimentación

- Conecta una batería de 9V o fuente externa al motor driver L298N para alimentar los motores.
- Asegúrate de que el GND de la batería esté conectado al GND del motor driver y del Arduino.


Paso 5: Programación

- Abre el IDE de Arduino y escribe el siguiente código para controlar el movimiento del robot:

// Definir los pines de control de los motores

int motor1Pin1 = 2; // IN1

int motor1Pin2 = 3; // IN2

int motor2Pin1 = 4; // IN3

int motor2Pin2 = 5; // IN4

void setup() {

  pinMode(motor1Pin1, OUTPUT);

  pinMode(motor1Pin2, OUTPUT);

  pinMode(motor2Pin1, OUTPUT);

  pinMode(motor2Pin2, OUTPUT);

}

void loop() {

}

  // Mover hacia adelante

  digitalWrite(motor1Pin1, HIGH);

  digitalWrite(motor1Pin2, LOW);

  digitalWrite(motor2Pin1, HIGH);

  digitalWrite(motor2Pin2, LOW);

  delay(2000); // Mantener el movimiento por 2 segundos

  // Detener el robot

  digitalWrite(motor1Pin1, LOW);

  digitalWrite(motor1Pin2, LOW);

  digitalWrite(motor2Pin1, LOW);

  digitalWrite(motor2Pin2, LOW);

  delay(2000); // Pausa de 2 segundos

2.Sube el código a tu placa Arduino.


Paso 6: Prueba

- Verifica que todo esté correctamente conectado y que la alimentación esté activa.
- Prueba el robot, y debería avanzar hacia adelante durante 2 segundos y luego detenerse. Puedes modificar el código para que haga más acciones como girar o retroceder.
